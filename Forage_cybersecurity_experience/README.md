## Forage Cybersecurity Experience - Commonwealth Bank Introduction to Cybersecurity Job Simulation

This folder contains the work I completed as part of the Commonwealth Bank Introduction to Cybersecurity Job Simulation on Forage. The program introduced key security concepts through practical, scenario-based tasks.

---

### Included Documents

#### **Penetration Testing Report**

A write-up analysing vulnerabilities across several simulated web challenges, including exploitation methods and recommended mitigations.

#### **Incident Response Analysis**

An assessment of a simulated malware/phishing incident, outlining identification, containment, eradication, and recovery steps.

#### **Passphrase Security Infographic**

A simple, user-focused infographic explaining the importance of strong passphrases and how to create them.
